# Gym-Analytics-Dashboard
Project Overview -

This project is a Gym Analytics Dashboard built using Excel and Power BI.
It analyzes gym membership, attendance, and revenue to provide actionable business insights for improving member engagement and revenue.

Features -

Track total members, active members, and revenue

Visualize membership trends by Age, Gender, and City

Analyze attendance patterns by Workout Type and Month

Track revenue by Membership Type and Payment Method

Interactive Power BI dashboard with KPI cards, charts, and slicers
